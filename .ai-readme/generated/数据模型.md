## 数据转换

### API 响应 → UI 模型

Repository 层负责将 API 响应转换为 UI 友好的数据类：

```kotlin
// Repository 示例
suspend fun getFiles(path: String): Result<List<FileItem>> {
    return runCatching {
        val response = fileApi.getFiles(path = path)
        response.data.files.map { apiFile ->
            FileItem(
                path = apiFile.path,
                name = apiFile.name,
                isDirectory = apiFile.type == "dir",
                size = apiFile.size,
                modifiedTime = apiFile.mtime,
                type = apiFile.type,
                thumbnailUrl = DsmApiHelper.getThumbnailUrl(apiFile.path)
            )
        }
    }
}
```

## 缓存策略

### Coil 图片缓存

```kotlin
// DSMApplication.kt
override fun newImageLoader(): ImageLoader {
    return ImageLoader.Builder(this)
        .okHttpClient { DsmApiHelper.imageClient }
        .memoryCache {
            MemoryCache.Builder(this)
                .maxSizePercent(0.25) // 25% 内存
                .build()
        }
        .diskCache {
            DiskCache.Builder()
                .directory(cacheDir.resolve("image_cache"))
                .maxSizeBytes(512L * 1024 * 1024) // 512MB
                .build()
        }
        .build()
}
```

### 会话自动重登录

当检测到会话过期时：
1. 尝试使用存储的凭据重新登录
2. 成功后更新 API 客户端会话
3. 失败则通知 UI 跳转登录页
