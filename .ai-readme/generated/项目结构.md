## 入口文件

### 1. DSMApplication.kt
应用级入口，负责：
- 初始化 `SettingsManager`
- 初始化 `DsmApiHelper` 和 OkHttpClient
- 异步加载会话状态
- 配置 Coil 图片加载器

```kotlin
@HiltAndroidApp
class DSMApplication : Application(), ImageLoaderFactory {
    override fun onCreate() {
        super.onCreate()
        instance = this
        SettingsManager.init(this)
        DsmApiHelper.initOkHttpClient()
        // 异步初始化
        ProcessLifecycleOwner.get().lifecycleScope.launch {
            SettingsManager.migrateIfNeeded()
            DsmApiHelper.loadSessionAndInit()
        }
    }
}
```

### 2. MainActivity.kt
主 Activity，负责：
- 设置 Jetpack Compose 内容
- 处理启动生物识别认证
- 初始化导航图 (`DSMNavHost`)
- 监听会话过期事件

```kotlin
@AndroidEntryPoint
class MainActivity : FragmentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        enableEdgeToEdge()
        setContent {
            DSMTheme {
                val navController = rememberNavController()
                DSMNavHost(navController = navController)
            }
        }
    }
}
```
