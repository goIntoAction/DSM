## 响应格式

### 通用响应结构

DSM API 响应统一格式：

```json
{
  "success": true,
  "data": { ... }
}
```

或错误响应：

```json
{
  "success": false,
  "error": {
    "code": 106,
    "errors": [
      {
        "code": "error_core_session_timeout",
        "message": "会话已过期"
      }
    ]
  }
}
```

### 错误码定义

**通用错误码** (`DsmApiHelper.getApiErrorMessage()`):

| 错误码 | 含义 |
|--------|------|
| 100 | 未知错误 |
| 101 | 账户不存在 |
| 102 | 账户已禁用 |
| 103 | 密码错误 |
| 104 | 权限不足 |
| 105 | 服务不可用 |
| 106 | 会话超时 |
| 107 | 会话中断 |
| 109 | 会话不存在 |

**登录错误码** (400-417):
- 400: 凭据错误
- 401: 账户已禁用
- 403: 需要 OTP
- 409: 会话过期
- 410: 账户已锁定

**文件操作错误码** (1000-1009):
- 1000: 文件未找到
- 1001: 文件夹未找到
- 1002: 文件已存在
- 1006: 权限不足
- 1007: 存储空间不足

**下载站错误码** (4000-4003):
- 4000: 任务未找到
- 4001: 任务创建失败
- 4002: 无效下载链接

## 鉴权机制

### 会话管理

**登录流程**：
1. POST `/webapi/auth.cgi` 发送凭据
2. 服务器返回 `sid` (Session ID) 和 `synoToken`
3. 客户端保存 `sid`、`cookie`、`synoToken` 到 `SettingsManager`
4. 后续请求通过 `SessionInterceptor` 自动添加会话信息

**会话存储**：
```kotlin
// SettingsManager.kt
val sid: StateFlow<String>
val cookie: StateFlow<String>
val synoToken: StateFlow<String>
val host: StateFlow<String>
```

### 会话过期处理

**检测机制**：
```kotlin
// DsmApiHelper.kt
private val SESSION_EXPIRED_CODES = setOf(106, 107, 109, 409)

private fun isSessionExpired(code: Int): Boolean = code in SESSION_EXPIRED_CODES
```

**自动重登录**：
1. 检测到会话过期错误码
2. 尝试使用存储的凭据重新登录
3. 重登录失败则发送 `sessionExpiredEvent`
4. UI 层监听事件后跳转登录页

```kotlin
// 会话过期事件
val sessionExpiredEvent: SharedFlow<Unit> = _sessionExpiredEvent.asSharedFlow()

// UI 层监听
LaunchedEffect(Unit) {
    DsmApiHelper.sessionExpiredEvent
        .onEach {
            navController.navigate(DsmRoute.Login) {
                popUpTo(0) { inclusive = true }
            }
        }
        .launchIn(this)
}
```

### Token 存储

| Token | 存储位置 | 用途 |
|-------|----------|------|
| sid | DataStore / Cookie | 会话 ID |
| synoToken | DataStore | CSRF 防护 |
| _sid | 查询参数 | 备用会话 ID |

## 文件上传/下载 URL

### 文件下载 URL

```kotlin
fun getDownloadUrl(path: String): String {
    val sessionId = getSessionId()
    return "${baseUrl}/webapi/entry.cgi?" +
        "api=SYNO.FileStation.Download&" +
        "version=2&method=download&" +
        "path=${URLEncoder.encode(path, "UTF-8")}&" +
        "mode=open&_sid=$sessionId"
}
```

### 照片缩略图 URL

```kotlin
fun getPhotoThumbnailUrl(photoId: Long, size: String = "m"): String {
    val apiName = if (dsmVersion >= 7) "SYNO.Foto.Thumbnail" else "SYNO.Photo.Thumbnail"
    return "${baseUrl}/webapi/entry.cgi?" +
        "id=$photoId&" +
        "size=\"$size\"&" +
        "api=\"$apiName\"&" +
        "method=get&" +
        "version=1&_sid=$sid"
}
```

## API 调用示例

### Retrofit 接口定义

```kotlin
// AuthApiRetrofit.kt
interface AuthApiRetrofit {
    @GET("webapi/auth.cgi")
    suspend fun login(
        @Query("api") api: String = "SYNO.API.Auth",
        @Query("version") version: Int = 7,
        @Query("method") method: String = "login",
        @Query("account") account: String,
        @Query("passwd") password: String,
        @Query("session") session: String = "DSM",
        @Query("enable_syno_token") enableSynoToken: Int = 1
    ): ApiResult<AuthResponse>
}
```

// AuthRepository.kt
suspend fun login(host: String, account: String, password: String): Result<Unit> {
    return runCatching {
        val response = authApi.login(account = account, password = password)
        if (response.success) {
            // 保存会话
            SettingsManager.saveSession(...)
        } else {
            throw ApiException(response.error.code)
        }
    }
}
